期中数学分析

2023年5月14日 · 阅读需 5 分钟

21题 这道题的解题思路，于公司的位置为坐标轴原点，向东为正，向西为负，小张不停地在数轴上移动（利用数轴地知识来解题）。画个草图或者表格：

这里采用 蛮力原则 + 表格思维

  1. 利用题目条件，根据表格，小张今天最终点位置为x=-3,不能回到公司。
  2. 今天总共走了：2+5+6+5+10+7+2 = 37
  3. 从表格来看，小张离公司最远的是8.

分析：这道题难度不大，尤其是第二题没有理解对。

22题 这个利用方程的思路，假设先安排x人进行工作，由已知条件列出方程： 总共工作量是40工时。

x*4 + (x+2)*8 = 40

简化：

12x + 16 = 40

得：

x = 2

答：略

23题 思路这个运算为：a^2 - ab

  1. 3^2 - 3*(-1) = 9+3=12
  2. 根据（）运算优先级，先算：
   3^2 - 3*4 = -3

再利用-3，计算：

   （-2）^2 - （-2）*（-3） = 4 - 6 = -2

总结： 这道题不应该错，列出式子，不负数没有注意。

24题 这道题，如果不了解，先用如何解决数学题的特殊原则 已知条件： （1）. a1=1,a2=3 （2）. a1+a2+a3+a4=a2+a3+a4+a5=a3+a4+a5+a6=...=an-3+an-2+an-1+an 解： 利用已知条件(2)，特殊原则：

a1+a2+a3+a4=a2+a3+a4+a5
a2+a3+a4+a5=a3+a4+a5+a6
a3+a4+a5+a6=a4+a5+a6+a7
a4+a5+a6+a7=a5+a6+a7+a8

得到： a1 = a5 a2 = a6 a3 = a7 a4 = a8 通过特殊原则，发现规律得出一般规则

a1=a5=a9=a13=...=a(4n+1), n = 0,1,2
a2=a6=a10=a14=...=a(4n+2),n = 0,1,2
a3=a7=a11=a15=...=a(4n+3),n = 0,1,2
a4=a8=a12=a16=...=a(4n),n = 1,2

这个数列是以4为循环的数，即an = a(n+4), n = 1,2,3,...

  1. a1=1,a2=3,并且a3=5，所以： a5=a(41+1)=a1 = 1, a2022 = a(4505+2)=a2 = 3

  2. 由前面的条件得，a7 = a3, a8=a4，且 a1=1,a2=3,a3=5,

|4 - a3 - a4|=|4 - 5 - a4| = 10

得： a4 = 9

S = a1+a2+a3+a4 + a5+a6+a7+a8 + ...+ a2017+a2018+a2019+a2020

由已知条件（2）：

a1+a2+a3+a4=a5+a6+a7+a8=...+a2017+a2018+a2019+a2020

所以：

S = （a1+a2+a3+a4）*505 = 18*505=9090

总结： 这道题 需要有一定抽象能力，所以要学会 利用特殊原则到一般原则。

25题 第一道题： 这道题要利用两点间A 其坐标点a,B其坐标点b的距离公式：

AB=|a - b|

  1. 利用两点间的距离公式：｜3-（-2）｜=5

  2. 假设C的新坐标为c，根据条件｜c - (-2)｜=2*|-4-(-2)|=4 得：

   ｜c+2｜= 4

根据去绝对值的知识点，得： c = 2， 或者 c = -6 两个条件都满足，所以c向左移动 1 或者 9 个单位。

第二题：

  1. 根据条件，A是向负方向移动，由当前的位置（-4）向左移动，所以A的新坐标为
-4 - at

同A的分析，B的新坐标为：

-2 - 2t

同A的分析，只不过C是往正方向移动，其新坐标为：

3 + 5t

根据1.的分析，利用两点间的距离公式， B于C的距离： d1 = |3+5t-(-2-2t)|=|5+7t|=5+7t A于B的距离： d2 = |-4-at-(-2-2t)|=|(a+2)t-2|(要考虑去绝对值) 这里由于 t >=1, (a+2)*t - 2 > 0，所以： d2 = |-4-at-(-2-2t)|=|(a+2)t-2| = (a+2)* t -2 

2*d1 - 3*d2 = 10+14t - 3*((a+2)*t - 2)
= 10+14t - 3*(a+2)*t + 6
= 16+(14-3a-6)*t

如果不随时间t的变化而变化，隐含的条件：

(14-3a-6) = 0

得： a = 8/3 这时：

2*d1 - 3*d2 = 16

总结： 1、两点间的距离公式。 2、利用代数式的知识表示坐标点。 3、利用一元函数 y = a * t + C (常量)，恒为常量C，则a = 0

这道题之前有讲过，看来掌握不是很熟。
