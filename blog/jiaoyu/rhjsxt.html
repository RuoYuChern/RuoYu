如何解数学题

2023年5月14日 · 阅读需 17 分钟

如何思维​

逆向思维 逆向思维法是指从事物的反面去思考问题的思维方法。逆向思维常常会这样思考： 从结果反向去寻找条件和解决方法。比如：

  1. 司马光砸缸救人： 如何救人？救人的最终结果是避免小孩被水溺死。那么如何避免被水溺死呢？如果把缸砸了，水流掉了是不是就可以避免这个问题呢？所以司马光就采用砸缸的方式。
  2. 草船借箭：周瑜提出让诸葛亮在十日之内赶制十万支箭的要求，那么如何完成任务？目标是向周瑜提供十万支箭，至于如何提供不重要，可以自己制造、可以去买、可以去抢、也可以去借。自己制造、买，浪费成本和时间，那么可以去抢吗？抢谁的？不好抢。可以去借吗？如果是借，如何借？所以就有草船借箭的行动，最终达成目标。
  3. 曹冲称象：目标是要冲出大象的重量，大象太大了目前的称无法称，那该采用什么方法？所以采用对比的方法，采用浮力的原理进行对比最终称出大象的重量。

正向思维 正向思维，就是人们在创造性思维活动中，沿袭某些常规去分析问题，按事物发展的进程进行思考、推测，是一种从已知进到未知，通过已知来揭示事物本质的思维方法。比如：

  1. 司马光砸缸救人：旁边有石头（已知条件），石头可以砸缸（已知条件），缸被砸破了，水就流出来了，小孩也因此得救了。
  2. 草船借箭：提前知道过几天大雾（已知条件），曹操多疑（已知条件），雾天能见度低、易于迷惑人。那么在大雾天用草船假扮军队去袭击曹操，曹操的士兵会把草船当作军队，从而放箭（推理），那么曹兵放的箭收集起来不就可以完成任务了吗？
  3. 曹冲称象：已知条件 大象太重、目前的称无法称出大象的重量。只能采用对比的方法进行，必须拿一堆可以分割的东西和大象进行对比。可以对比的东西有石头、木材，石头比较常见就采用石头。那要拿多少的石头进行对比呢？可以采用浮力的原理，确定有多少块石头（这里有一点逆向思维）。然后这些石头的重量加起来就是大象的重量。

逆向思维的特点，从结果找条件。正向思维的特点是从条件进行推理得出结果。逆向思维从事物的反面去思考，比如对某一个事物，大家都从正对方向（好）去思考，逆向思维却思考事物相反面（不好、缺点），一般有效的思维方式 逆向思维和正向思维相结合。

如何解决数学题​

  1. 理解题目：已知数据（条件）是什么，未知量是什么？条件有可能满足吗？条件是否满足以确定未知量？或者它不够充分？或者多余？或者矛盾？画一张图，引入适当的符号。
  2. 找出已知数据和与未知量之间的联系。如果找不到直接的联系，要考虑辅助题目（辅助线、辅助条件），最终应该得到一个解题方案。
  3. 执行解决方案，检查每个步骤，能清楚地看出这个步骤是否正确，能证明它是否正确。
  4. 检查已经得到的解答。

例子一

  x^4 - 13x^2 + 36 = 0

这个题目已知条件是整个方程，未知条件是x为多少。从题目可以看到x^4是x^2的平方，可以引入新的变量y = x^2。 解题过程： 令：

   y = x^2

原方程变为：

   y^2 - 13y + 36 = 0

因式分解：

   （y - 4）*(y - 9) = 0

也可以用求根公式 得：

   y = 4 或 y = 9

最终得

   x = +(-)2 或 x = +(-)3

例子二 光明小学有学生900人，其中女生的4/7与男生的2/3参加了课外活动小组，剩下的340人没有参加。请问男女生个各多少人？ 已知条件：

  1. 男生 + 女生 = 900.
  2. 4/7*女生 + 2/3*男生 = （900 - 340）= 660. 未知量：
  3. 男生多少？
  4. 女生多少？ 从当前的条件来看，只要知道男生的数量，那么女生的数量就可以直接算出。因此引入辅助变量男生=x. 解： 令 男生的数量为x
   (900 - x)*4/7 + x*2/3 = 660

得x = 480，那么 女生的数量： 900 - 480 = 420.

例子三 2只羊、3匹马和4头牛每天共吃草143千克；1只羊、4匹马、2头牛每天共吃108千克。1匹马每天吃草多少千克？ 已知条件： 简化：一只羊每天吃的草重记为羊，一匹马每天吃的草重记为马，一头牛每天吃的草重记为牛。

2*羊 + 3*马 + 4*牛 = 143 
1*羊 + 4*马 + 2*牛 = 108

未知条件： 1: 一匹马每天吃的草重。 从这里，我们可以发现已知条件无法直接满足未知量，但是可以发现条件(1) 和 条件（2）羊和牛的数量倍数关系都是2， 2: 引入辅助条件：2*羊 + 8*马 + 4*牛 = 216 解： 一只羊每天吃的草重记为a，一匹马每天吃的草重记为b，一头牛每天吃的草重记为c。 由条件得：

  1. 2*a + 3*b + 4*c = 143
  2. 1*a + 4*b + 2*c = 108
  3. 由（2） 2 得``2a + 8b + 4c = 216``
  4. 由（3）-（1）得5*b = 73 最终得： b = 14.6.

数学思维​

算两次原理​

我们在计算出某些东西的数目，但却是用完全不同的方法去计算。

举例一 高斯算： 1+2+3+4+5+...+ 100 = ？ 高斯利用算两次的原理，采用完全不同的方法计算：

   1+2+3+4+...+ 100
   100+99+98+97+...+1

上下两行逐个相加，（1+100）、（2+99），...(100+1)都是101，总共有100个。 解： 令：

   S = 1+2+3+4+...+ 100
   S + S = （1+100）+ (2+99)+ (3+98)+ ...+(100+1)
         = 101*100 = 10100

所以

         S = 101*100/2 = 5050

举例二 求 S=1/2 + 1/2^2 + 1/2^3 + 1/2^4 + 1/2^5，分子很有规律是2的多次方（1、2、3、4、5）。根据分子是2的多次方，利用算两次原则

S = 2*S - S = 2（1/2+1/2^2 + ...+1/2^5） - (1/2+1/2^2 + ...+1/2^5)
S = 1+1/2 + ...+1/2^4 - (1/2+1/2^2 + ...+1/2^5)
S = 1 - 1/2^5

狄利克雷原理（抽屉原理）​

完全的无序是不可能的。如果N+1 个对象要任意放到N个格子里，至少会有一个格子放了两个物品。

举例子：假如有3本数放到2个抽屉里，无论怎么放，都会有一个抽屉至少有两本书。 这个原理一般用于推论一些数论相关的问题。

相反原则（反证法）​

先假设某个断言的反面是对的，然后通过无懈可击的逻辑推导，得出与假设事实矛盾的结论，以此来证明原本的断言是否对。 反证法具有以下的逻辑结构：

  1. 叙述 A（演绎推理）
  2. 条件（或假设）非A。
  3. 推论（没有明显的矛盾）。
  4. 推论（显然矛盾）
  5. 关于叙述A的结论。

举例子：没有最大的自然数。 反证法：

  1. 叙述 A：没有最大的自然数。
  2. 非A： 假设有最大的自然数，记为N。
  3. 推论（没有矛盾）： 最大的自然数N 加上 1为：N+1.
  4. 推论： 显然 N+1 > N
  5. 矛盾： 假设不成立，即没有最低的自然数是对的。

归纳原则（归纳法）​

为了证明一堆有序对象当中的全部东西皆具有某些性质，可以先证明第一个东西有此性质，然后证明，若其中一个东西具有该性质，则下一个东西也具有此性质。

举例子 猜想：

1^ 3 + 2^3 + ... + N^3 = ?

采用特殊原则进行规律： 1.当N=1，1^3 = 1 = 1^2 2.当N=2，1^3 + 2^3= 1 + 8 = 9 = 3^2 3.当N=3，1^3 + 2^3 + 3^3 =36 = 6^2 4.当N=4，1^3 + 2^3 + 3^3 + 4^3= 10^2 5.当N=5，1^3 + 2^3 + 3^3 + 4^3 + 5^3= 225 = 15^2 6.当N=6，1^3 + 2^3 + 3^3 + 4^3 + 5^3 + 6^3= 441 = 21^2 7.当N=7，1^3 + 2^3 + 3^3 + 4^3 + 5^3 + 6^3 + 7^3= 784 = 28^2

发现规律从N=1，...,6,7 这个式子的和为如下的数的平方： 1、3、6、10、15、21、28的平方， 所以这里要发现规律，算出1、3、6、。。。。28 与N=1，...,7的关系。 引入辅助变量，当： N=1, a1=1. N=2, a2=3. N=3, a3=6. N=6, a6=21. N=7, a7=28. 这里要发现规律： 当N>=2，有an - an-1 = n 所以： a1 = 1 a2 - a1 = 2 ... an - an-1 = n 所以： a1 + (a2-a1) + ...+ （an - an-1） = 1+2+。。。+n=（n+1）n/2 即： an = (n+1)n/2 当N=1，也成立。 所以猜测：

1^ 3 + 2^3 + ... + N^3 = （(n+1)*n/2）^2

正式利用规律法证明： 当N = 1，

1^ 3  = （(1+1)*1/2）^2 = 1

成立。 假设m=K时，也成立：

1^ 3 + ...+K^3  = （(K+1)*K/2）^2

那么m= K+1 时，则

1^ 3 + ...+K^3 +（K+1）^3 = （(K+1)*K/2）^2 + (K+1)^3
 = （(K+1)*K/2）^2 + (K+1)^3 = (K+1)^2*(K^2/4+K+1)
 = (K+1)^2*(K^2 + 4K + 4)/4=(K+1)^2*(K+2)^2/4=((K+1)*(K+1+1)/2)^2

也成立

一般化原则​

解决一般问题时，可不可以先删除去一些条件或是改变一些约束条件，然后再把求得的解运用在眼前的特俗情况。

举例一 证明：

   S= 1/1^2 + 1/2^2 + 1/3^2 + ... + 1/n^2 <= 2

利用一般化原则，改变约束条件：

   an = 1/n^2 <=1/(n*(n-1)), n>=2

所以：

S <= 1 + (1-1/2) + (1/2 - 1/3) + ... + (1/(n - 1) - 1/n)
S <= 2 - 1/n <= 2

举例二 比较：A=29/62 和 B=293031/626160的大小。 观察下A和B，发现B的分母和分子很有规律，所以利用一般化原则： 令C为：

C = 292929/626262 < B 
C = 29*10101/(62*10101)=29/62 = A.

举例三 已知S=1/（1/1991 + 1/1992 + ...+1/2000）的整数部分。 像这种题目，我们一般用不等式进行一般化原则，改变约束条件去头尾：

1/1991+... + 1/1991 > (1/1991+...+1/2000)>1/2000 + ...+1/2000

得：

10/1991 > (1/1991+...+1/2000)>10/2000

得：

1991/10 < S < 2000/10

所以S 的整数部分是 199. 举例四 符号F表示一种运算，它对一些数的运算定义为：

F（1）= 0,F（2）= 1,F(3)=2,F(4)=3,F(5)=4,...
F（1/2）= 2,F（1/3）= 3,F(1/4)=4,F(1/5)=5,...

根据F的定义，利用一般化规则有：

F(n) = n -1, n >=1,n是整数
F(1/n) = n, n=2,n 是整数。

所以：

F(1/n) - F(n-1) = n - (n-1-1) = 2

即：

F（1/2015）- F（2015-1）= 2

特殊原则​

解决问题时，先看特殊情况，然后从特殊情况的结果推广到一般情况的解。 参考归纳法中，利用特殊数字进行规律发现。

举例一 求： 111...111 总共1000个1除于6的余数是多少？ 解题思路： 这么多个111...111，很显然不好处处理，我们利用 特殊原则进行规律发现：

  1. 11 除于 6余数是5.
  2. 111 除于 6余数是3.
  3. 1111 除于 6余数是1.
  4. 11111 除于 6 余数是5. 从这里我们可以发现规律，从最开始的11除于6余数为5，再跟下一个数组成51 除于6余数3，3再跟下一个1组成31除于6，余数5.因此除了第一个1之外，3位一个循环，那么1000个减去1个之外，还有999个1，第999个1刚好是循环结束，因此 111...111，1000个1 除于6 余数1. 即除了第一个1除外，位数除于3，余数为1，其除于6余5，余数为2，其除于6余3，余数为0，其除于6余1.

变化原则​

可以通过控制改变问题的某些层面，从新的角度来观察，对原本的问题有更深入的理解，进而解开问题。 在上述举的例子中，都有变化原则的利用。

转化观点原则​

解题时可以从目标往起点反向进行，然后再翻转思考方向。先逆向思维，再正向思维。 参看前面对正向思维和逆向思维的讨论。

蛮力原则（穷极法）​

通过试遍所有可能的解法来解题。

举例一 一堆苹果，2个2个数剩1、3个3个数剩2、4个4个数剩3，5个5个数剩4，6个6个数剩5，则这堆苹果至少有（）个？ A：59， B：57，C：55，D：53. 首先根据排除法，不能被5，也不能被3整除，排除B、C。由于题目说至少多少个，则思路： 1、从53、59 根据蛮力法则一个一个地试，最终答案：59. 这题的思路：先排除后蛮力。
