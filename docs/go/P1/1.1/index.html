Hello Go

学习一门语言，不管是C 还是C++，亦或是Java， 都是通过写Hello world，来了解：

  1. 编程环境搭建。
  2. 代码结构。
  3. 代码编译、执行。
  4. 设置代理， 解决依赖下载慢的问题。

1.1.1环境搭建​

1.1.1.1Go安装​

  1. 下载Go，登陆https://go.dev/dl/，根据自己的操作系统、CPU类型，下载相应的版本。Intel 32是x86、Intel 64是x86-64。

  2. 安装Go，Linux 系统下载格式为tar.gz 文件，具体方法参考：https://go.dev/doc/install，或者Baidu。

  3. 设置环境变量GOPATH、GOBIN。其中GOPATH是日常开发的根目录，GOBIN 是go命令的安装路径。

    MacOS 设置

    打开MacOS的命令终端：

    cd ~ // 进入用户主目录
    vi ~.bash_profile
    # 输入i：进入 输入模式
    # 输入 如下的代码
    export GOROOT=/usr/local/go
    export GOBIN=$GOROOT
    export PATH=$PATH:$GOBIN
    export GOPATH=/Users/ruoyu/gospace
    # 如果临时生效，即当前命令窗口生效，可以不用设置.bash_profile文件中
    # 按esc键，并输入:wq 退出编辑模式。
    source ~.bash_profile #使其立即生效
    go env # 查看是否配置成功

    MacOS 由于命令窗口是zsh，则要设置在~.zshrc 文件中，其它方法一样，最后输入source ~.zshrc 使其立即生效。

    Linux系统

    环境变量设置方法参考 MacOS 写到~.profile。go的安装路径也是/usr/local/go。

    Windows

    1. 右键我的电脑，选择【属性】选项，进入系统窗口。

    2. 单击【高级系统设置】，打开【系统属性】窗口。

    3. 单击【环境变量】按钮，打开【环境变量】窗口。

    4. 在【系统变量】区域，单击【新建】按钮，打开【新建系统变量】窗口，在该窗口中的

      变量名: 输入 GOPATH

      变量值：输入go的安装目录，一般是C:\Program Files\Go。

      最后 点击确定。

      同样的GOPATH 也重复4的步骤。

    5. 返回【系统变量】区域，选择Path，点击【编辑】进入【编辑环境变了】，点击【新建】在变量列表出现可编辑行，输入：

      %GOPATH%/bin

    6. 点击确定。

  4. 设置代理。

    打开命令行界面，输入如下：

    # 启用 Go Modules 功能
    go env -w GO111MODULE=on

    # 配置 GOPROXY 环境变量，以下三选一

    # 1. 七牛 CDN
    go env -w  GOPROXY=https://goproxy.cn,direct

    # 2. 阿里云
    go env -w GOPROXY=https://mirrors.aliyun.com/goproxy/,direct

    # 3. 官方
    go env -w  GOPROXY=https://goproxy.io,direct

1.1.1.2 集成开发环境搭建​

目前Go的IDE 只要有两种：Golang 和 Visual Studio code。由于Goland 是试用版，本书不解释如何安装，这个相对比较容易些。本书的代码都通过Visual Studio code进行开发，因此本书介绍下，如何通过Visual Studio code 搭建Go的开发环境。

步骤：

  1. 下载并安装Visual Studio code，下载地址：https://code.visualstudio.com/

  2. 安装Go 扩展，打开Visual Studio code，如图进行操作：

  3. 更新Go工具，如图进行操作： 进入按钮选择界面，单击【命令面板】进入： 输入: GO， 选择 Install/Update Tools 进入： 全选进行安装 代码提示、自动补全、静态检查、测试等功能。

  4. 具体参考：https://learn.microsoft.com/zh-cn/azure/developer/go/configure-visual-studio-code

  5. Visual Studio code 中文化，参考步骤2 Go扩展，在扩展界面，搜索 Chinese，选择简体中文进行安装，并重新打开。

1.1.2代码编写​

万事具备之后，开始进行 ”Hello world“编写。

1.1.2.1 模块初始化​

​ 开始写Hello world 之前，先初始化模块。模块的作用是对依赖进行管理。初始化步骤：

  1. 打开命令行终端.

  2. 安装如下顺序输入命令：

    cd Documents/code/gospace  # 这个路径就是之前设置的 GOPATH，日常开发路径
    mkdir gomicroguid          # 本书参考代码路径
    cd gomicroguid/            # 进入代码路径
    go mod init tao.com/gomicroguid # 创建模块，这时会输出go: creating new go.mod: module tao.com/gomicroguid
    ls                         # 查看是否产生了go.mod文件，这时终端输出go.mod

  1. 模块命名：

    可以： xxx.com/gomicroguid， 也可以gomicroguid。

1.1.2.2 编写​

  1. 通过集成环境Visual Studio code 创建新文件hello.go，如图：

  1. 输入代码：

    // 包名
    package main

    // 引入依赖
    import "fmt"

    /**
    * 定义sayHello的函数
    * 参数：world
    */
    func sayHello(world string){
       fmt.Printf("Hello %s!!!", world)
    }

    // 程序入口函数
    func main() {
      // 输出
        fmt.Println("Hello Go!!!")
        fmt.Println("Hello world !!!")
    }

    代码参考： https://github.com/RuoYuChern/GoMicroGuid 下的hello.go

1.1.2.3编译​

通过命令

cd Documents/code/gospace/gomicroguid  # 进入目标gomicroguid
go build hello.go #编译

可以通过集成环境底下【终端】打开命令终端。

1.1.2.4 运行​

运行有两种方式：

  1. 通过命令终端，点击集成环境底下【终端】，出现命令终端之后，输入如下：

    ./hello
  1. 通过集成环境工具栏，点击【运行】下的【启动调试】进行调试

1.1.3 Go 代码规范​

通过上文Hello.go 代码的编写，读者可以看出Go代码的一些规范。这节主要讲Go的一些规范。

1.1.3.1 代码文件规范​

// 包名   -- 这个是单行注释
package main
/**
* 多行注释
* 引入依赖
* 多依赖引入
**/
import (
  "abc"
  "xyz"
)
//import "fmt" -- 如果是依赖引入

/**
* 函数定义: 通过关键字func 
* 函数名称:sayHello
* 参数：world，类型 string
**/
func sayHello(world string){
  
}

/**
* 结构体声明（类）:type 名称 struct
* 结构体名称: Person
**/
type Person struct{
   age int32;
   name string;
}

/**
* 接口声明:type 名称 interface
**/
type People interface{
  Work();
  Say();
  Think();
}

/**
* 程序入口
* main 函数只能在 package main中定义
*/
func main() {
  // 声明变量
  var world string
    fmt.Println("Hello Go!!!")
    sayHello("world")
}

总结：

  1. 一个程序只能有一个入口函数（main）。
  2. 入口函数main 定义在package main 中。
  3. 一个package xxx的实现可以单文件，也可以多文件，即多个文件的package 可以都是xxx。
  4. 通过import 可以引入 一个或者多个非main包。
  5. 声明规范：
// 声明函数： func name(参数列表)（返回列表）
func name(v1 string, v2 int32,...)(string,int32,...)
// 声明类：type name struct 
type name struct{}
// 声明接口：type name interface
type name interface{}
// 声明变量 var name 类型
var name string
// 声明变量并赋值 name:=xxx
//声明一个整型变量age
age:=10

1.3.2 预留关键字​

Go预留的关键字是读者在学习Go遇到的功能所用的关键字，这些关键字不能被在代码中声明的名称使用。比如func 声明定义函数的关键字，那么类似如下的语句就是错的：

var func string

以下的列表就是Go的一些预留关键字

intint8int16int32int64uintuint8uint16
uint32uint64uintptrfloat32float64truefalsebool
functypestructinterfacebytenewcopycase
ifelsebreakforrangeimportmapdefault
changotogoswitchpackageleniotareturn
stringprintlncontinueconstvarselectappend
