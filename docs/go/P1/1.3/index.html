Go 数据类型

程序操纵的变量，是有数据类型的。那么Go中有几种数据类型呢？本节的学习目的：

  1. Go的数据类型及其类型转换。
  2. 打印不同类型的变量。
  3. 了解数组和切片
  4. 了解map

1.3.1 数据类型​

Go有如下的的几种数据类型：

  1. 基本数据类型（原生数据类型）：整型、浮点型、复数型(相比C/C++，Go新增的)、bool型、字符串、字符。
  2. 派生数据类型：数组、切片、映射、结构体、接口、通道、指针、函数。

1.3.1.1 整型​

整型存储的是整数：0，1，2,....

Go中的整型 如下：

注： 整型的取值范围，可以通过math包的相关定义来获得。

func pintInt() {
    fmt.Printf("int8: [%d, %d)\n", math.MinInt8, math.MaxInt8)
    fmt.Printf("uint8: [%d, %d)\n", 0, math.MaxUint8)
    fmt.Printf("int16: [%d, %d)\n", math.MinInt16, math.MaxInt16)
    fmt.Printf("uint16: [%d, %d)\n", 0, math.MaxUint16)
    fmt.Printf("int32: [%d, %d)\n", math.MinInt32, math.MaxInt32)
    fmt.Printf("uint32: [%d, %d)\n", math.MinInt8, math.MaxInt8)
    fmt.Printf("int64: [%d, %d)\n", math.MinInt64, math.MaxInt64)
    fmt.Printf("uint64: [%d, %d)\n", 0, uint64(math.MaxUint64))
}

其它的打印格式，参考：https://pkg.go.dev/fmt#pkg-types

1.3.1.2 浮点型​

浮点型存储的是小数，Go浮点型：

类型字节数取值范围说明
float324<=math.MaxFloat3232位的浮点数
float648<=math.MaxFloat6464位的浮点数

打印float

func pintFloat() {
    var f32 float32 = 10.123456
    var f64 float64 = 1000.123456
  // 原样打印
    fmt.Printf("float: [%f, %f)\n", f32, f64)
  // 只保留4位小数
    fmt.Printf("float: [%.4f, %.4f)\n", f32, f64)
}
// 输出
//float: [10.123456, 1000.123456)
//float: [10.1235, 1000.1235)

1.3.1.3 bool型​

bool 类型用于逻辑判断，比如if 判断。Go中bool类型的值只能是常量: true 和 false.

func printBool() {
    var isHigh bool = false
  // bool 类型值取反 !
    var isLow bool = !isHigh
  // 用判断表达式给bool类型赋值
    var isEqual bool = (3 == 5)
  // bool 类型打印:%t
    fmt.Printf("isHigh:%t, isLow:%t, isEqual:%t\n", isHigh, isLow, isEqual)
}

1.3.1.4 字符型​

字符组成是字符串的元素，Go中的字符有两种：

类型字节数说明
byte1表示utf-8字符串单个字节的值，uint8的
别名类型。
rune4表示单个unicode字符，int32的别名类型。

学习到字符，就有必要了解下Go的转义字符，这些转义字符跟其它语言（C/C++/Java）一样。

转移字符说明
\r回车符return，返回到行首
\n换行符，直接跳到下一行行首。
\tTAB键
\'单引号
\"双引号
\\ 斜杠
func printChar() {
  // 声明 byte
    var a byte = 'A'
  // 声明rune 赋值中文
    var b rune = '学'
  // 声明c 并赋值'，需要转义
  var c byte = '\''
  // 字符打印，采用\t、\"，\n 等转义符号
  fmt.Printf("a: '%c'\tb:\"%c\"\tc:%c\n", a, b, c)
  // 新一行
  fmt.Printf("next line\r")
}
//输出
//a: 'A'  b:"学"  c:'
//next line%

1.3.1.5 字符串​

Go字符串，表达一串由多个字符组成的内容，可以用两种方式表示：

  1. 双引号\"" 单行字符串（单行）。
  2. 反引号``多行（跨行），反引号中的所有字符都不会被编译，当作字符串中的内容。
func printStr() {
  // 声明一个单行字符串
    var s string = "Hello Go"
  // 声明一个多行字符串
    var m string = `Hello Go,
     func(){
        fmt.Printf("%s",s)
     }`
    fmt.Printf("s:%s\n", s)
    fmt.Printf("m:%s\n", m)
}

1.3.1.6 复数型​

复数是Go中新增的，用于表示数学中的复数，比如1+2i，3-4i。

数据类型字节数说明
complex648由float32 类型声明的实部和虚部复数
complex12816由float64 类型声明的实部和虚部复数
func printComplex() {
  // 通过内建函数complex 声明初始化
    var c64 complex64 = complex(10, 20)
    var c128 complex128 = complex(100, 200)
  // %v 原样打印复数，real 获取实部，imag 获取虚部
    fmt.Printf("c64:%v, real: %f, imag:%f\n", c64, real(c64), imag(c64))
    fmt.Printf("c64:%v, real: %f, imag:%f\n", c128, real(c128), imag(c128))
}

//c64:(10+20i), real: 10.000000, imag:20.000000
//c64:(100+200i), real: 100.000000, imag:200.000000

1.3.1.7 类型转换​

Go 语言中不支持 隐式类型转换，比如C/C++ 中整型可以隐式转换成浮点类型。Go的类型转换格式：

  1. T(表达式)，T 是目标转换类型。
  2. 常见的类型转换是： int{8,16,32,64}/uint{8,16,32,64}， float{32,64} 等直接转换，以及整型转为字符串。
  3. 整型转字符串，不是将整数转为相应的数字字符串，而是转成该数字对应的ASCII值对应控制符号。
  4. 字符串不能通过类型转换转为整型，会报错。
  5. 注意： 在编码中，我们经常遇到 数字与字符串之间的转换，不是用类型转换。
func typeChange() {
    var i8 int8 = 10
    var i32 int32 = int32(i8)
    var f32 float32 = float32(i32)
  // 按照ASCII字符集转换，ASCII 值 10 是 LF
    var str string = string(i32)
    var f64 float64 = 10.345
    var i64 int64 = int64(f64)
  // 大写 A对应的ASCII值
  var a int8 = 65
  var as string = string(a)
    fmt.Printf("i8:%d, i32:%d, f32:%f, str:%s, f64:%f, i64:%d\n", i8, i32, f32, str, f64, i64)
  fmt.Printf("a: %s\n", as)
}
/*
i8:10, i32:10, f32:10.000000, str:
, f64:10.345000, i64:10
a: A
*/

1.3.1.8 数组及切片​

数组： 是指相同类型的元素组成的固定长度的序列：

  1. 数组元素数据类型都相同。
  2. 数组的长度是固定的，不可改变。
  3. 声明方式： var 变量名 [数组长度] 类型
  4. 数组可以是多维的：var 变量 [长度][]...[长度]类型
  5. 数组是值类型的（指针会详细分析）。
  6. 数组下标是从零开始，即数组元素访问： 数组[i]， 0=<i < 长度

由于Go中的数组长度是固定的、不可改变，但是在很多场景，数组的长度是不可预知的，比如在Restful 接口中返回的元素列表。

Go 中的切片就是为了弥补这种缺陷，本质上来讲切片是一种指针（数组指针）：

  1. 指针，指向序列中指定的开始位置。切片是引用类型的。
  2. 通过cap(切片) 可以获取切片的容量（最大的元素个数），通过len(切片)可以获取切片的长度（已有的元素个数）。
  3. 声明方式： var 切片名称 []类型， 声明未初始化其值是nil。
  4. make(t Type, size ... int) 函数参数： 第一个参数类型 []Type，第二参数是len，第三个参数cap。
  5. copy(dst []Type, src []Type)：将src 拷贝到dst，拷贝到元素个数是min(len(dst), len(src)) 。
func printSlice() {
    //声明但未初始化
    var ar1 [4]int
    fmt.Printf("ar1 len: %d, cap:%d\n", len(ar1), cap(ar1))
    for i := 0; i < 4; i++ {
        ar1[i] = (i + 1)
    }
    // 声明并初始化
    var ar2 = [4]int{1, 2, 3, 4}
    // 声明并初始化
    var ar3 = [...]int{1, 2, 3, 4}
    for i := 0; i < cap(ar1); i++ {
        fmt.Printf("ar1[%d] = %d, ar2[%d] = %d, ar3[%d] = %d\n", i, ar1[i], i,
            ar2[i], i, ar3[i])
    }

    // 通过截取数组来获取
    var sl1 []int = ar1[:]
    fmt.Printf("sl1 len: %d, cap:%d\n", len(sl1), cap(sl1))
    for _, v := range sl1 {
        fmt.Printf("v:%d\t", v)
    }
    fmt.Printf("\n")
    // 声明未初始化
    var sl2 []int
    fmt.Printf("sl2: value:%p len: %d, cap:%d\n", sl2, len(sl2), cap(sl2))
    // 赋值
    for i := 0; i < 4; i++ {
        sl2 = append(sl2, i+1)
    }
    for _, v := range sl2 {
        fmt.Printf("v:%d\t", v)
    }
    fmt.Printf("\n")
    fmt.Printf("sl2: value:%p len: %d, cap:%d\n", sl2, len(sl2), cap(sl2))
    for i := 0; i < len(sl2); i++ {
        fmt.Printf("%d\t", sl2[i])
    }
    fmt.Printf("\n")
    // 采用make 函数初始化，这里初始化一个长度和容量都为3的
    var sl3 []int = make([]int, 3)
    fmt.Printf("sl3: value:%p len: %d, cap:%d\n", sl3, len(sl3), cap(sl3))
    // 采用copy 函数拷贝
    l := copy(sl3, sl2)
    fmt.Printf("copy len:%d\n", l)
    for i := 0; i < len(sl3); i++ {
        fmt.Printf("%d\t", sl3[i])
    }
    fmt.Printf("\n")
}
/*
ar1 len: 4, cap:4
ar1[0] = 1, ar2[0] = 1, ar3[0] = 1
ar1[1] = 2, ar2[1] = 2, ar3[1] = 2
ar1[2] = 3, ar2[2] = 3, ar3[2] = 3
ar1[3] = 4, ar2[3] = 4, ar3[3] = 4
sl1 len: 4, cap:4
v:1     v:2     v:3     v:4
sl2: value:0x0 len: 0, cap:0
v:1     v:2     v:3     v:4
sl2: value:0xc00011c020 len: 4, cap:4
1       2       3       4
sl3: value:0xc000118018 len: 3, cap:3
copy len:3
1       2       3
*/

1.3.1.9 map​

​ 在实际情况中，我们经常遇到一种场景： 通过某个值去获取其关联的值。这种情况下，就要用到字典map。map 是一种key-value映射关系数据类型。

  1. map 是无序的，底层是由Hash 来实现的。
  2. map的key 是唯一的，相同的key 指向同一个value，且其类型必须是可以比较的，即支持== 或者!=的类型，所以bool、整型、浮点型、字符串、数组可以作为key。切片和函数则不能作为key。
  3. map 的value 是可以任意数据类型的。
  4. map 是一种引用类型。
  5. map 声明： var 变量名 map[key类型]value类型，也可以通过make函数声明并初始化。
func pintMap() {
    // 声明初始化map, 未初始化是nil 指针，不能操作
    var m1 map[string]int = make(map[string]int)
    fmt.Printf("m1 len:%d", len(m1))
    // 添加元素
    m1["A"] = 10
    fmt.Printf("m1 len:%d\n", len(m1))
    // 访问元素: 返回(value, bool)
    v, ok := m1["B"]
    if ok {
        fmt.Printf("value:%d\n", v)
    }
    m1["B"] = 11
    // 遍历
    for k, v := range m1 {
        fmt.Printf("m1 key:%s, value:%d\n", k, v)
    }
    // 声明并初始化 map
    var m2 = map[string]int{
        "A": 10,
        "B": 11,
    }
    // 删除元素
    delete(m2, "B")
    for k, v := range m2 {
        fmt.Printf("m2 key:%s, value:%d\n", k, v)
    }
}
/*
m1 len:0m1 len:1
m1 key:A, value:10
m1 key:B, value:11
m2 key:A, value:10
*/

1.3.1.10 结构体​

​ 结构体是用来定义复杂的数据类型，其声明格式：

type Name struct{
   m1 Type1
   m2 Type2
   m3 Type3
}

上述声明的变量，其成员值初始化为相应类型的默认值，结构体是值类型。

例子：

// 定义人的结构体
type Person struct {
    name string
    age  int
    sex  string
}

func printPerson() {
  // 声明变量，其成员初始化为相应类型的默认值
    var p1 Person
  // 打印结构体的原来值
    fmt.Printf("p1: %v\n", p1)
  // 给成员变量赋值
    p1.age = 10
    p1.name = "Ry"
    p1.sex = "M"
  // 打印结构体的原来值
    fmt.Printf("p1: %v\n", p1)
  // 声明 并初始化成员
    var p2 = Person{
        name: "RR",
        age:  10,
        sex:  "M",
    }
    fmt.Printf("p2: %v\n", p2)
  // 通过深度拷贝p2声明一个结构体变量p3
  p3 := p2
}
/**
p1: { 0 }
p1: {Ry 10 M}
p2: {RR 10 M}
*/

1.3.2 通用打印​

格式化打印通常使用fmt 包，不管什么样的数据类型，都可以由如下的通用打印进行打印，其它格式的打印在各个章节都有例子。

通用打印：

格式说明
%v值的默认格式表示。
%+v%v的功能类似，如果是结构体，则采用类似Json格式打印（会加上成员名称）。
%#v值的Go语法表示。
%T值的类型Go 语法表示。
func printGen() {
    f1 := 10.45
    var p2 = Person{
        name: "RR",
        age:  10,
        sex:  "M",
    }
  // 按照通用格式打印
    fmt.Printf("f1: %f, %v, %+v, %#v, %T\n", f1, f1, f1, f1, f1)
    fmt.Printf("p21: %v, %+v, %#v, %T\n", p2, p2, p2, p2)
}
/*
f1: 10.450000, 10.45, 10.45, 10.45, float64
p21: {RR 10 M}, {name:RR age:10 sex:M}, main.Person{name:"RR", age:10, sex:"M"}, main.Person
*/

注：

如果需要格式化输出并返回字符串，则采用fmt.Sprintf(fmt, ...)
